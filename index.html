Auto Patcher

A simple APK to quickly apply English translation for the game Ragnarok Origin.

Game patching is not supported by Gravity

Require Android version 4.1 or above.

Auto Patch

  1. Download the App.
  2. Run the Patcher.
  3. Choose your desired server.
  4. Tap the Patch button and wait for it to patch your Ragnarok Origin app.

Manual Patch

  1. Download the files needed for manual patch.(download link provided below)
  2. Go to your Phone storage.
  3. Go to "Android" folder.
  4. Go to "data" folder.
  5. Go to "com.gravity.ragnarokorigin.aos" folder.
  6. Go to "files" folder.
  7. Go to "HotUpdate" folder.
  8. Paste the two(2) downloaded files on the HotUpdate folder.

Start the game and Enjoy!
